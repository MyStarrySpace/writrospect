## Key Findings for App Design

### What the Science Supports

| App Feature | Scientific Support |
|-------------|-------------------|
| **Specific tasks** | Specific goals outperform vague ones (d = .42-.80) |
| **Context in strategies** | Implementation intentions work via if-then planning |
| **Progress dashboard** | Self-monitoring improves outcomes (multiple meta-analyses) |
| **No streak penalties** | Missing one day doesn't reset habit formation |
| **Private by default** | Public commitment can suppress commitment-making |
| **User autonomy** | Choice enhances intrinsic motivation |
| **Goal + obstacles** | MCII (wish + obstacle + plan) is effective |

### What to Avoid

| Anti-pattern | Why |
|--------------|-----|
| Punitive features | Threats/pressure diminish intrinsic motivation |
| Rigid timelines | Habit formation varies 18-254 days |
| Forced public sharing | Can reduce commitment-making |
| Gamification without care | Extrinsic rewards can undermine intrinsic motivation |
